HPC-VS Binge Drinking Overview

This folder contains the analysis file, manuscript figures, model outputs, and a limited subject-level HPC-VS connectivity export for the HPC-VS binge drinking analyses.

Folder structure:
- Data/: HCP Subject IDs and derived left/right HPC-VS limbic striatum connectivity values only.
- Figures/: exported 300 dpi manuscript figures.
- HPC_Ordinal_ManuscriptOrdered_Permutation_DFBETAS_Output/: CSV outputs from ordinal models, permutation checks, DFBETAS sensitivity checks, age sensitivity, and diagnosis follow-ups.
- HPC_Final.Rmd / HPC_Final.pdf: analysis source and rendered report.

The full HCP behavioral, demographic, diagnostic, alcohol-use, and other analysis variables are not included. Open-access HCP data require ConnectomeDB registration and acceptance of the HCP data-use terms; restricted HCP variables require a separate Restricted Data application and case-by-case approval. Start here: https://db.humanconnectome.org and https://www.humanconnectome.org/study/hcp-young-adult/data-use-terms

The segmented HPC-VS values are the proportion of streamlines in a given hippocampal (HPC) voxel connected to ventral striatum (VS; limbic striatum from Tziortzi et al., 2014), divided by streamlines that reached VTA, vmPFC, or VS. See Elliott et al. (2024), "In vivo structural connectivity of the reward system along the hippocampal long axis": https://onlinelibrary.wiley.com/doi/abs/10.1002/hipo.23608
